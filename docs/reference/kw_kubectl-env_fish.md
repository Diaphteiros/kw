## kw kubectl-env fish

Generate a fish script that points the KUBECONFIG env var to the kubeconfig for the current kw session

### Synopsis

Generate a fish script that points the KUBECONFIG env var to the kubeconfig for the current kw session.

The script can be called manually or added to your shell's profile to automatically set the KUBECONFIG env var when you start a new shell.

```
kw kubectl-env fish [flags]
```
