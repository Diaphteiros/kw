## kw repeat

Switch to the last used configuration (cross-session)

### Synopsis

Switch to the last used configuration (cross-session).

This is basically an alias for 'kw history load 0 --global'.
It requires a history depth of at least 1 (meaning the history must not be deactivated).

```
kw repeat [flags]
```
